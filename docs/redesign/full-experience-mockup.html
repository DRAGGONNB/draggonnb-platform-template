🧭 How navigation works (best of both worlds)

  1. Click a sidebar item (e.g. "Content Studio") — navigates to that section's Overview page (the smart-landing for that module)
  2. Hover over a sidebar item — flyout appears with all sub-pages, click any to jump straight there
  3. Once on the page, horizontal tabs at the top let you switch between Overview / Social / Email / Sequences / Drafts / Analytics
  4. The Easy ⇄ Advanced toggle (top-right of each page) controls level of detail per tab — Phase 11 pattern, extended platform-wide as Autopilot ⇄ Hands-on
👉 Try it: hover over Content Studio or CRM in the sidebar to see the flyout. Click the item itself to navigate to its main page.

When you might use each

⚡ Hover flyout
Power users · jump 2 clicks deep without losing context
🖱 Click sidebar
First-time users · land on overview, see everything at once
📑 Tabs on page
Already inside a module · switch sub-features without going back to sidebar
